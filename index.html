Perguntas Frequentes

Como funciona o licenciamento e os 3 dias grátis? +

O programa gera um número de identificação único (ID) para o seu computador. Se você abrir o programa conectado à internet (Internet), ele reconhecerá seu ID automaticamente e liberará 3 dias de teste grátis. Após esse período, basta nos enviar o seu ID pelo WhatsApp para adquirir sua licença definitiva.

Preciso de internet (Internet) para usar o programa? +

Não! O Teleprompter e o Apresentador de Slides funcionam de forma totalmente autônoma sem conexão com a internet (Offline). Caso você não tenha internet no momento da instalação, basta copiar o ID gerado na tela, nos enviar, e nós geramos uma chave que você digita para liberar o programa localmente.

O que acontece se o computador desligar enquanto eu digito? +

O sistema possui uma tecnologia de salvamento automático (Auto-Save). A cada 3 segundos, tudo o que você escreve é guardado com segurança em uma pasta oculta de backup no seu computador. Ao reabrir o programa, seu texto estará exatamente onde você parou.

O software recebe atualizações? +

Sim! O programa conta com um motor inteligente de atualizações. Quando há uma nova versão disponível, ele realiza o download automático mantendo todas as suas preferências e licenças perfeitamente configuradas e substituindo o arquivo antigo de forma segura.
